

\# VIDEX Selenium Automation Bot



Automatisiert das Ausfüllen des VIDEX-Formulars (Kurzaufenthalt) mittels Selenium.



\## Features

\- Automatisches Ausfüllen von Formularfeldern

\- PDF-Download nach Abschluss

\- Konfigurierbar über .env
